그리디 알고리즘은 탐욕법으로 말 그대로 탐욕적으로 문제를 푸는 것이다.

탐욕적이게 문제를 푼다? 직관적으로 와닿진 않는다. 탐욕적으로 문제를 푼다는 것은 <u>현재 상황에서 가장 좋은 것만 고르는 것이다.</u>

동전이 무한히 있다고 가정하고 최소의 동전의 갯수로 거슬러 주는 알고리즘을 구현했다.
---

<pre><code>
#include <stdio.h>

int main() {
  int price = 770;
  int change = 0;
  int coin[] = { 500,100,50,10 };
  
  for (int i = 0; i < sizeof(coin) / sizeof(int); i++) {
    change += price / coin[i];
    price = price % coin[i];
  }
  printf("거스름돈 동전 갯수는: %d", change);
}</code></pre>
